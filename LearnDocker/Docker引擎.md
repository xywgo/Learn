**Docker引擎**是用来运行和管理容器的核心软件。通常人们会简单地将其代指为 Docker 或 Docker 平台。

**Docker 引擎**由如下主要的组件构成：Docker 客户端（Docker Client）、Docker 守护进程（Docker daemon）、containerd 以及 runc。它们共同负责容器的创建和运行。
